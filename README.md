# 📄 Landing Page – Escritório de Advocacia

Landing page institucional desenvolvida com foco em **responsividade (mobile-first)**, **clareza de comunicação** e **experiência do usuário**, voltada para um escritório de advocacia com atuação nas áreas previdenciária, trabalhista, consumerista e de família.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (vanilla)

---

## 📱 Conceitos Aplicados

* ✅ Mobile First
* ✅ Responsividade completa (mobile, tablet e desktop)
* ✅ UI moderna e profissional
* ✅ Copywriting jurídico adaptado ao Código de Ética da OAB
* ✅ Performance e carregamento otimizado

---

## 🎯 Funcionalidades

* Hero com imagem de fundo e overlay
* Botão de contato direto via WhatsApp
* Sessão "Sobre" com layout dividido (imagem + texto)
* Sessão de especialistas com fotos e descrições
* Listagem de demandas em formato de tópicos interativos
* Cards de áreas de atuação com efeito visual
* Animações suaves em hover
* Footer com informações institucionais
